# Cyber-Twin Router + IoT + Blockchain + Cloud (2026 Master's Project)

**100% software-only • No hardware • Windows + WSL2 ready**

## Quick Start (Windows)
1. Install Docker Desktop + WSL2
2. Double-click `setup-windows.ps1` (PowerShell)
3. Open http://localhost:5000

## Features
- Digital Twin Router (FRR)
- Cowrie Honeypot (mimics Cisco router)
- Simulated IoT Sensors (MQTT)
- Ganache Blockchain (immutable logs)
- Flask Cloud Dashboard + Grafana

Tested on Windows 11 + WSL2 – April 2026
